# py_hook_virtual_machine

# 解决方案
> pyhook 项目通过虚拟机+ pyhook 方案的方式, 通过虚拟鼠标单击的方式, 触发虚拟机中的支付宝咻咻以实现自动化的刷咻咻的功能

## 环境配置要求
- windows 系统 (xp+ , 2g memory at least )
- android 虚拟机（bluestack ect.）
- python运行环境（python , pywin32 , pyhook）

## 基本功能要求
1. 获取支付宝触发区域的位置
2. 模拟鼠标单击事件
3. 可通过键盘快捷方式控制pyhook开始与暂停
4. 能够快速打包部署到其他设备上,应用

## 拓展功能
- 动态变化每次鼠标单击事件时间段,以防止被识别为外挂
- 刷到红包后能够提示已经收到宝贝（声音, 邮件等方式）
- 能够智能识别捕获支付宝咻咻中心区域.实现启动自动单击的方式


#  开发阶段
- 捕获当前鼠标的位置信息
- 捕获鼠标快捷键
- 模拟鼠标单击事件
- 鼠标单击事件间隔动态随机处理


# 软件架构设计
> 软件的主线程提供给hook作为监控键盘输入, 工作线程和主线程通过全局变量的方式来相互通信, 主线程控制工作线程的开始和暂停工作, 确保软件能够可控的范围内.
